Euclidian Distance¶

Euclidean distance is the most common way to measure the straight-line distance between two points in a Euclidean space. Think of it as how you'd measure the distance between two locations on a flat map using a ruler.

Example: Cereal Nutritional Similarity

Let's use the cereal nutritional data you've been working with to illustrate Euclidean distance. Imagine we want to find out how "similar" two cereals are based on just two of their nutritional facts: Sugars and Protein (g).

Suppose we have the following data for two hypothetical cereals (let's assume they are standardized values, so their scales are comparable):

Cereal A:

Sugars: 1.5 Protein (g): 0.8 So, Cereal A as a point is (1.5,0.8) Cereal B:

Sugars: 0.5 Protein (g): 1.2 So, Cereal B as a point is (0.5,1.2)

​ So, the Euclidean distance between Cereal A and Cereal B is approximately 1.077. This single number quantifies how similar or dissimilar these two cereals are based on their sugar and protein content. A smaller distance indicates greater similarity.

3. Utility function to calculate the Euclidian Distance¶

In [26]:
def euclidean_distance(point1, point2):
    """
    Calculates the Euclidean distance between two n-dimensional points.
    Assumes points are pandas Series or numpy arrays.
    """
    point1 = np.array(point1)
    point2 = np.array(point2)
    return np.sqrt(np.sum((point1 - point2)**2))

Interpretation¶

Standardization is essential for Euclidean distance because it prevents features with larger numerical ranges from disproportionately dominating the distance calculation. Without it, a small percentage difference in a feature like "Practice Questions Completed" (e.g., values ranging from 0-500) could outweigh a much more significant percentage difference in "Hours Studied" (e.g., values ranging from 0-20), simply because the squared difference of larger numbers will be much greater. By scaling all features to a common range (like a mean of 0 and standard deviation of 1), standardization ensures that each feature contributes equally to the distance, reflecting the true underlying similarity based on the pattern of values across all features rather than just their raw magnitudes.
